Problem Summary

There is an issue in healthcare management, especially in predicting the length of stay for patients being admitted into hospitals. Being able to predict length of stay is crucial for managing hospital resources and improving hospital and healthcare efficiency. Our primary objective is to develop a predictive model that can estimate the length of stay for a given patient. We hope to use regression, decision trees, and clustering to find the most effective predictors. By solving this issue, hospitals will be able to properly allocate bed space, fix staffing needs, and properly utilize equipment.

Data We Are Using

For our goal of predicting the length of stay for each hospital patient, we will be utilizing a Kaggle Dataset. AV: Healthcare Analytics II. This dataset contains a separate training data file, which includes patient, hospital, and length of stay data on a case-by-case basis. It also contains a feature dictionary that provides descriptions and possibly data types for the features in the training data. Additionally, it includes testing data that contains the features related to patients and hospitals but lacks length of stay data.

How We Plan to Address the Problem

One way that we can address the problem is by utilizing a linear regression model that helps determine the possible independent variables that have a significant relationship to the length of stay (dependent variable). Additionally, we can use a heatmap to showcase a visual representation of how strongly correlated or not these variables are to each other. We can also utilize a CART (Classification and Regression Tree) that can simplify the data, making it easier to interpret as well as break down the data to understand its significance and overall relationships. Furthermore, we plan to utilize classification trees to separate groups based on characteristics that would ease the process of prediction when it comes to the length of stay.
